将该文件添加到暂存区

添加一个或多个文件到暂存区：

git add [file1] [file2] …

添加指定目录到暂存区，包括子目录：

git add [dir]

添加当前目录下的所有文件到暂存区：

git add .

将暂存区内容添加到本地仓库中（提交）

提交暂存区的指定文件到仓库区：

git commit ‘file1‘ ‘file2‘ … -m ‘xxx’

-a 参数设置修改文件后不需要执行 git add 命令，直接来提交

git commit -a

将add和commit合为一步

git commit -am ‘xxx’

合并上一次提交（用于反复修改）

git commit –amend -m ‘xxx’

ps： xxx 为备注信息

删除文件

将文件 xxx 从暂存区和工作区中删除：

git rm xxx

如果删除之前修改过并且已经放到暂存区域的话，则必须要用强制删除选项 -f。
强行从暂存区和工作区中删除修改后的 xxx 文件：

git rm -f xxx

把文件从暂存区域移除，仍保留在当前工作目录中 –cached

git rm –cached xxx
